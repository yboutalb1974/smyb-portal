Come aggiornare il portale
01
Netlify — drag & drop
Vai su app.netlify.com, entra nel tuo sito, clicca "Deploys" e trascina l'intera cartella con tutti i file HTML. Il portale si aggiorna in 30 secondi mantenendo lo stesso URL.
Trascina la cartella → Deploy automatico
02
GitHub Pages — push
Carica i file aggiornati nel repository GitHub. Se hai GitHub Desktop, basta trascinare i file nella cartella del repo e fare commit + push. Il sito si aggiorna in 1-2 minuti.
git add . → git commit → git push
03
Aggiungere un nuovo strumento
Crea il file HTML nella stessa cartella, poi aggiungi una nuova card in questo index.html copiando il pattern esistente. Ogni tool è un file HTML autonomo — nessuna dipendenza esterna.
nuovo-tool.html → nuova card in index.html
